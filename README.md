## 🎯 Sobre o Projeto
Sistema ERP desenvolvido para modernizar a gestão de compras públicas. O diferencial é a integração de um módulo de Inteligência Artificial capaz de gerar e editar documentos burocráticos (DFD, ETP, TR) com base nos dados dos pedidos.

## 🚀 Tecnologias
* **Backend:** Python, FastAPI, SQLAlchemy
* **Banco de Dados:** PostgreSQL (Planejado)
* **IA:** Integração com LLMs (OpenAI/Local)
* **Arquitetura:** MVC / Repository Pattern
